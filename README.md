# CloudAppi Exam

## Requirements
- npm
- node
- Postgres

## How to Install
1. Clone repo
  ```
  git clone https://github.com/Fasulicious/CloudAppi-Exam.git
  ```
2. Replace credentials in .env (`export $(cat .env)`)
3. Install dependencies
  ```
  npm install
  ```
4. Create user and database

## Run
```
npm run deploy
```
